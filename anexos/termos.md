
# Glossário de termos

**Ata:** Registo formal das decisões e ações de uma reunião.  
**Requisito:** Necessidade ou condição a ser satisfeita pelo projeto.  
**Relatório semanal:** Documento que resume o progresso do projeto numa semana.  
**Branch:** Linha de desenvolvimento separada utilizada para organizar alterações.  
**Main:** Branch principal onde é integrado o trabalho validado.  
**Merge:** Processo de integrar alterações de uma branch noutra.
