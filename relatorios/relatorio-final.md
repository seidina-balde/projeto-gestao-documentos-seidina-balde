
# Projeto Gestão de Documentos

Este repositório organiza a documentação de um projeto fictício, incluindo planeamento, atas, relatórios, comunicação e anexos. O objetivo é garantir uma estrutura clara, controlo de versões e registo de decisões, requisitos e progresso.

## Estrutura de pastas

- planeamento
- atas
- relatorios
- comunicacao
- anexos

## Como contribuir

1. Criar sempre uma nova branch diária a partir da main atualizada.
2. Fazer commits pequenos e com mensagens claras.
3. Solicitar validação antes de integrar alterações na branch main.
